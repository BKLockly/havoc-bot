# havoc-bot
> havoc上线提示插件，支持钉钉，企业微信



## 使用

1. 克隆仓库到Havoc项目的`/data/extensions/`下
2. 在配置文件中填好相应的webhook
3. 也可以在scripts -> scripts manager -> load script -> 选择py脚本 -> 顶部工具栏点击进行设置，保存即可
